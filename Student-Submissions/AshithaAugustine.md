Project Description:
This project analyzes US Natural Disaster Declarations from 1953 to 2025 using Power BI.
The analysis focuses on identifying disaster trends, state-wise patterns, and storm activity insights.

The repository includes:

- Data Import Process
- Data Cleaning Documentation
- Dimension Tables
- Data Model
- DAX Measures
- Power BI Interface
- Dashboard Screenshots
- Project Documentation (README)

Dashboard Highlights:

Dashboard 1: From Texas to Tennessee – Unveiling US Disaster Patterns
- Total Disasters
- Total States
- Total Incident Types
- Total FEMA Declarations
- Disaster trends by state and year
- Peak disaster year identified (2020)
- Severe storms identified as the most common incident type

Dashboard 2: US Storm Activity – State and Seasonal Trends
- Total Storm Incidents
- Average Storms per Year
- Monthly Storm Trend Analysis
- State-wise Storm Distribution
- Identification of peak disaster months
